# Weather App 2

Русскоязычное статическое приложение погоды без API-ключей.

## Возможности

- Поиск городов и подсказки через Open-Meteo Geocoding.
- Текущая погода и прогноз на 7 дней через Open-Meteo Forecast.
- Почасовой прогноз на 24 часа.
- Кнопка "Погода рядом со мной" через геолокацию браузера.
- Избранные города и история поиска в `localStorage`.
- Фото города через Wikipedia/Wikimedia.
- Локальные часы выбранного города.
- Курсы валют и калькулятор.
- Темная и светлая тема.
- PWA manifest и service worker для установки приложения.

## Запуск

Откройте `index.html` в браузере.

Для проверки PWA используйте локальный сервер, например:

```bash
python -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Файлы

- `index.html` - разметка приложения.
- `styles.css` - стили и мобильная адаптация.
- `app.js` - логика приложения.
- `manifest.webmanifest`, `sw.js`, `icon.svg` - PWA.
- `AGENTS.md`, `HANDOFF.md` - правила и журнал передачи между агентами.
